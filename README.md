# MSIL-WFH-PORTAL

A full-stack Work From Home management portal built for Maruti Suzuki employees.  
The system helps employees manage WFH requests, leave applications, attendance records, company notices, holidays, notifications, and profile details from one centralized dashboard.

## Features

- Employee login and signup
- Corporate email validation
- Interactive employee dashboard
- WFH request management
- Leave application and leave history
- Attendance overview
- Holiday and calendar integration
- Notifications panel
- Profile dropdown and employee details
- Company rules and policy pages
- Team members section
- Reports section
- Responsive UI
- Backend REST APIs
- MongoDB database integration

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Tailwind CSS
- Material UI Icons
- Recharts
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- CORS
- Dotenv
- Nodemon
